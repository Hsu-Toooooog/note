# Source组

1. ### Power source

   - AC: 交流电源
   - DC: 直流电源

# Basic组

1. ### RESISTOR：电阻

2. ### CAPACITOR： 电容

3. ### INCDUCTOR：电感

4. ### CAP_ELECTROLIT：电解电容

5. ### VARIABLE_RESISTOR：可变电位器(两脚，不提供滑动端接口)

6. ### POTENTIOMETER：可变电位器(三脚，含滑动端接口)

# Transistors组(晶体管组)

9014、9013可以用2N2222,2SC1815、2SC945代替（NPN管）

9012可以用2SA1015、2SA733、2SA954代替（PNP管）
